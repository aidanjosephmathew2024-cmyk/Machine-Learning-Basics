# Machine Learning Basics

This repository contains machine learning projects developed using Python.

## Projects

### Iris Classifier
- Implemented a custom K-Nearest Neighbors (KNN) classifier
- Trained and evaluated on the Iris dataset
- Generated predictions for flower species classification

### Handwriting Classifier
- Built a handwritten digit recognition model
- Used image preprocessing and classification techniques
- Evaluated model performance on test data

### Student Performance Prediction (Random Forest)
- Built a classifier to predict student academic performance
- Achieved 99.33% test accuracy
- Used Random Forest with Scikit-learn

## Technologies
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow/Keras
